## Install

Make sure that you have the Gatsby CLI program installed:
```sh
npm install --global gatsby-cli
```

And run from your CLI:
```sh
git clone git@github.com:saigowthamr/gatsbyblog-test.git
```

Then you can run it by:
```sh
cd gatsbyblog-test
gatsby develop
```
